إعدادات التكامل:

لربط تقويم جوجل، يرجى اتباع الخطوات التالية:

  1. اذهب إلى Google API Console.
  2. أنشئ مشروعًا جديدًا أو اختر مشروعًا قائمًا.
  3. قم بتمكين Google Calendar API.
  4. أنشئ بيانات اعتماد (OAuth 2.0 client IDs).
  5. أدخل Client ID و Client Secret أدناه.

لا توجد إعدادات محددة لهذا التكامل حاليًا.
